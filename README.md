# SauceDemo BDD Automation Framework

A professional automation suite built using **Java**, **Selenium WebDriver**, and **Cucumber** (Gherkin). This project demonstrates **Behavior-Driven Development (BDD)** principles by translating plain English requirements into automated test scripts.

## Key Features
*   **Gherkin Scenarios:** Human-readable test cases using `Given`, `When`, `Then`.
*   **Step Reusability:** Efficient use of a dictionary-style step definition layer.
*   **Data-Driven Testing:** Uses Scenario Outlines to test multiple user personas (standard, locked-out).
*   **Automated Reporting:** Generates clean execution logs via the TestRunner.

## Tech Stack
*   **Language:** Java 17
*   **Framework:** Cucumber with JUnit 4
*   **Tool:** Selenium WebDriver
*   **Build:** Maven
